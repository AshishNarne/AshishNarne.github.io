BingeBuddy

  • Many struggle to find the perfect movie to watch in their free time.
  • This C++ program can find the perfect movie from the IMDB Internet Movie Database using the IMDB V4 API.
  • The user is given options to organize movies by their Length, rating, number of ratings, year, title, director, and genre in either ascending or descending order.
  • This organizes the movies in the preferred order that the user would like, and the user also gets the option to choose specifically what topic they would like to look at, such as length or title..
  • This program also allows users to use both merge and shell sort algorithms to find the most optimal algorithm.
